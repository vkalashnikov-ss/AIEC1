# Session 16:  🏆 Reinforcement Learning with Verifiable Rewards

🎯 Learn to implement RLVR to reasoning performance in LLMs.

📚 **Learning Outcomes**

- Understand Group Relative Policy Optimization (GRPO) and how it differs from Proximal Policy Optimization (PPO)
- Learn how to design reward functions for verifiable tasks
- Learn how RL can further improve reasoning through iterative optimization

## 📛 Required Tooling & Account Setup

- No new tools or accounts are required

- Any OS (macOS, Linux, Windows). Unlike Session 15, this session needs no GPU.
